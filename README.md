# Little-Monkeys
app for parents to find events suited to their child's age and interests

## Technology Stack

### Front-end
* React
* React DOM
* React Router
* Webpack from scratch
* ESLint
* Jest

### Back-end
* Node/Express
* JSON Web Token (JWT), cookies, bcrypt
* Axios, Async/Await
* Travis CI

### Database
* Mongo DB

### Hosting
* Heroku
